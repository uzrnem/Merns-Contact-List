# node-vue-typescript-crud
Crud Starter using NodeJS, ExpressJS and VueJS written in TypeScript

Install Node Packages

```bash
npm run install
```

Run Development

```bash
npm run dev
```

Run Production

```bash
npm run prod
```

Vue JS :
* Seprate HTML, JS, SCSS files

Node JS :
* Seprate Controller, Router files
